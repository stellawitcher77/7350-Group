# AI Video Web Showcase

This folder contains four static website variants based on:

- the earlier review of AI video generation tools, application scenarios, advantages, and limitations
- the structure and writing prompts in `7350 网页文稿.docx`

Each variant includes five pages:

- `index.html`
- `tools.html`
- `scenarios.html`
- `controversies.html`
- `marketing.html`

Open `index.html` in this folder first to compare the four design directions.

## Variant list

- `template-01-editorial-motion` - Template 01 / Editorial Motion: Magazine-inspired, paper-toned, serif-led, calm but authoritative.
- `template-02-signal-grid` - Template 02 / Signal Grid: Control-room dashboard with electric contrast and sharp geometry.
- `template-03-sunlit-studio` - Template 03 / Blue Horizon: Airy light-blue and white interface with crisp information panels and a cleaner academic tone.
- `template-04-midnight-cinema` - Template 04 / Midnight Cinema: Moody cinematic panels with luxe contrast and poster energy.
